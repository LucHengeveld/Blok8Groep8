Extra information about the use of this application.

Click on the subject for more information.

The OR keywords are the keywords of which at least one has to be in the PubMed articles. The OR keywords can be added in the OR input field. The keywords have to be separated by commas. To add multiple OR statements click on the button 'Click to add more OR input fields'. These OR input fields can be removed by clicking on the button 'Click to remove the last OR input field' which removes the input field that has been added most recently. Only one OR field or AND field is mandatory.

The AND keywords are the keywords that all have to be in the PubMed articles. The AND keywords can be added in the AND input field. The keywords have to be separated by commas. Only one OR field or AND field is mandatory.

The NOT keywords are the keywords that are not allowed to be in the PubMed articles. The NOT keywords can be added in the NOT input field. The keywords have to be separated by commas. This field is not mandatory.

The gene names are the names of the genes that will be searched for in the PubMed articles. The gene names can be added in the gene names input field. The genes have to be separated by commas. If a value is added, it will only show this gene in the results table. This field is not mandatory.

A publication date can be given by clicking on the calendar icon and selecting a date or by typing the date in the input field. The chosen date is the earliest date for which PubMed articles will be searched. If there is no publication date given, all the results will be shown. When there are no results after a certain date, it will say "No results found".

The gene panel selection are the names of the gene panels that will be compared with the genes that have been found. The excluding of gene panels have to be separated by commas. The gene panels can be added in the gene panel input field. The gene panel file has to be an Excel file. The gene column in the excel file must be named "Symbol_HGNC", the gene synonyms column must be named "Aliases" and the genepanel column must be named "GenePanel". There will occur an error when it's not an Excel file.

An e-mail adress can be added for the NCBI tools incase it overloads. The e-mail can be added in the e-mail input field. This field is not mandatory, but if you are overloading NCBI they will send you an e-mail instead of blocking the IP-adress.

Results are shown after the button 'Search for PubMed articles' has been clicked. Firstly a list of all the specified filters is shown. After that a table will be shown with the found genes, the title of the PubMed article with a link to that article, the publication date and potentially the gene panels the gene is in. The article and genes with the highest relevance score will be shown on top. The results can be downloaded as a file by selecting one of the file extensions and clicking the button 'Save results'.
